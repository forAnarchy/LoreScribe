# LoreScribe
LoreScribe is a free, browser-based translation editor built for anyone who works on game localization.

Many games store their text deep inside complex binary files rather than plain readable documents. Translating them requires special tools, technical knowledge, and a lot of patience. LoreScribe tackles the most tedious part of that process: it displays original and translated text side by side, highlights untranslated lines in yellow and lines that are too long in red, and lets you download the corrected file in one click.
No installation. No sign-up. Open your file, translate, download.
Supports SF3, CSV, PO, XML, JSON, INI and plain text formats. Available in Turkish, English, Japanese, Chinese, Urdu, Hindi and Arabic.
